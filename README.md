# pl2303_tinygpsplus

TinyGPSPlus version of [examples/pl2303/pl2303_tinygps](https://github.com/felis/USB_Host_Shield_2.0/blob/master/examples/pl2303/pl2303_tinygps/pl2303_tinygps.ino).

Tested on the following environment.

- M5Stack CoreS3 SE
    - USB Module v1.2
- GR-M02U

## Upload

```sh
pio run -t upload
```

## Monitor

```sh
pio device monitor
```
